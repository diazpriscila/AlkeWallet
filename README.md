# Alke Wallet

Proyecto frontend desarrollado como parte del módulo de Frontend del bootcamp.

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery

## Funcionalidades
- Login de usuario
- Visualización de saldo
- Depósito de dinero
- Envío de dinero a contactos
- Visualización de movimientos
